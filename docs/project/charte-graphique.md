# Charte Graphique — Book Nils Beneville

## Palette
| Rôle | Valeur | Usage |
|------|--------|-------|
| Background | `#0a0a0a` | Fond principal |
| Card BG | `#151515` | Fond cartes/éléments |
| Text | `#f0f0f0` | Texte principal |
| Muted | `#888888` | Texte secondaire, labels |
| Accent | `#c9a84c` | Titres accent, CTA, hover |

## Typographie
| Rôle | Font | Weight |
|------|------|--------|
| Titres | Playfair Display | 400, 700 |
| Corps | Inter | 300, 400 |

## Ambiance
- Cinématographique, sombre
- Accent sur le mouvement et le corps
- Photos contrastées, légèrement désaturées au repos
- Animations subtiles (fade-in au scroll)

## Breakpoints
| Label | Valeur |
|-------|--------|
| Mobile | ≤ 480px |
| Tablet | ≤ 768px |
| Desktop | > 768px |

## Composants
- Hero plein écran avec gradient overlay
- Galerie en grille 3 colonnes (responsive)
- Bouton CTA : bordure accent, fond transparent, hover inversé
- Sous-titres projets en italique + metteur en scène

# Startup Kit

## One-Shot Kickoff Prompt
"Kickoff projet: initialise le startup pack complet.
1) Cree/valide les docs et logs obligatoires.
2) Assigne un agent responsable par asset + trigger de mise a jour.
3) Recommande les skills a embarquer (mandatory/recommended/optional) selon la stack.
4) Propose la premiere entree decision-log de demarrage et active le guardrail: pas de deploiement sans GO explicite."

## Mandatory Startup Assets
- `docs/project/decision-log.md`
- `docs/project/operating-rules.md`
- `docs/project/requirements-matrix.md`
- `docs/project/roadmap.md`
- `docs/project/glossary.md`
- `docs/project/architecture-standards.md`
- `docs/security/cybersecurity-baseline.md`
- `docs/security/robustesse-scalabilite.md`
- `.github/copilot-instructions.md`

## UI Projects Also Add
- `docs/project/frontend-patterns.md`
- `docs/project/charte-graphique.md`
- `design-audit`
- `design-harmony`

## Kickoff Customization (Required)
At startup, fill these files before development:
- `docs/project/architecture-standards.md` (stack constraints and approved patterns)
- `docs/project/roadmap.md` (phases and delivery plan)
- `docs/project/glossary.md` (canonical vocabulary)
- `docs/security/cybersecurity-baseline.md` (controls and policy)
- `docs/security/robustesse-scalabilite.md` (risk register)
- `.github/copilot-instructions.md` (project variables and governance fit)
